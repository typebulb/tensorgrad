tensorgrad samples

Small browser-side ML programs built on the tensorgrad library.

  • Digit Canvas (MNIST)

    MNIST classifier you can draw against in real time. Trains in the background while you watch; predictions update on every stroke. Exercises the widest slice of the API: polymorphic batch, dropout/no-dropout split, replaceModel, setOptimizerConfig, clipGradNorm, singleFlight.

  • MNIST CNN

    Two-block CNN trained from scratch on MNIST. Verifies the conv2d / maxPool2D / flatten path end-to-end on GPU. Status streams to the dev-server console.

  • Transformer learns addition

    Multi-layer transformer training from scratch on 2-digit addition. Full forward + backward + Adam compiled to WGSL via tensorgrad. ~1000 examples/sec.

  • MLP fits sin(x)

    Tiny 2-layer MLP regressor. Demonstrates the same library on a much smaller model — lets you see autograd, Adam, and the Module abstraction work in a different shape of problem.
