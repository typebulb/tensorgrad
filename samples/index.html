tensorgrad samples

Small browser-side ML programs built on the tensorgrad library.

  • Digit Canvas (MNIST)

    MNIST classifier you can draw against in real time. Trains in the background while you watch; predictions update on every stroke. Exercises the widest slice of the API: polymorphic batch, dropout/no-dropout split, replaceModel, setLR, clipGradNorm, singleFlight.

  • MNIST CNN

    Two-block CNN trained from scratch on MNIST. Verifies the conv2d / maxPool2D / flatten path end-to-end on GPU. Status streams to the dev-server console.

  • Flow-matching-tiny (MNIST)

    A small rectified-flow / flow-matching model trained from scratch on MNIST. Train for a few thousand steps then watch a digit emerge from noise as the sampler Euler-integrates the learned velocity field backward in time. Conv-stack model (no U-Net), learnable timestep embedding, in-graph noise generation.

  • NeRF-tiny (image INR)

    A 4-layer MLP fits a single image as an implicit function (x, y) → (r, g, b). Live reconstruction next to the target as the model trains. Demonstrates sinusoidal positional encoding (sin / cos at multiple frequency bands), without which the network can only fit the low-frequency part of the image. Upload your own image to swap targets.

  • CartPole REINFORCE

    Reinforcement learning: a tiny policy MLP learns to balance an inverted pendulum via discounted-return-weighted log-policy gradients. K parallel environments per rollout, fixed-length rollouts with episode-done masking. No supervised target — the gradient comes from the agent's own actions weighted by how well things went after taking them.

  • VAE (MNIST)

    Variational autoencoder with the reparameterization trick. Watch random samples become recognisable digits as training proceeds, then slide between two test digits in latent space to see the model's learned manifold morph one into the other.

  • Tic-Tac-Toe self-play

    A policy MLP plays itself; the gradient signal is just the terminal outcome of each game. Play the model while it trains and watch it get harder. Reaches optimal play (every game vs another optimal player ends in a draw) in a few hundred rollouts.

  • Transformer learns addition

    Multi-layer transformer training from scratch on 2-digit addition. Full forward + backward + Adam compiled to WGSL via tensorgrad. ~1000 examples/sec.

  • Makemore (name generation)

    Same transformer architecture as the addition demo, retargeted to Karpathy's makemore task: train on ~32K real first names, sample novel plausible names. The genuine ML payoff — outputs are defined by the data distribution, not by any specifiable algorithm.

  • MLP fits sin(x)

    Tiny 2-layer MLP regressor. Demonstrates the same library on a much smaller model — lets you see autograd, Adam, and the Module abstraction work in a different shape of problem.

  • KAN-tiny (per-edge splines)

    A 2-layer Kolmogorov-Arnold Network fitting y = sin(πx). Each edge of the network is a learnable cubic B-spline rather than a scalar weight; the per-edge curves are extracted via downloadParams and plotted live so you can watch the network discover a decomposition of the target.
