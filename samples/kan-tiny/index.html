KAN-tiny

A 2-layer Kolmogorov-Arnold Network fitting y = sin(πx). Each edge is a learnable univariate function (a cubic B-spline) rather than the usual scalar-weight-times-fixed-nonlinearity. The bottom grid plots every edge's spline; watch them evolve as the network discovers a decomposition of the target.

Architecture: two KAN layers 1 → 4 → 1. Each layer's output is Σ_i φ_{o,i}(x_i) + silu(x) @ W_res, where every φ_{o,i} is a cubic B-spline (8 coefficients, integer-spaced centers covering [-1, 1]). Adam, lr 5e-3, MSE loss, batches of 256. No new ops vs the rest of the library — the B-spline basis is a few where / abs / mul calls.
